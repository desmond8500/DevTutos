# [Astro](readme.md)

## Description

Framework Web

## Installation

```console
npm create astro@latest
npm run dev
```

## Variables

### Variable statique

```html
---
const name = "Astro";
---
<div>
  <h1>Hello {name}!</h1>  <!-- Outputs <h1>Hello Astro!</h1> -->
</div>
```
